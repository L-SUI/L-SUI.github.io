# nrm 安装以及使用

易于使用，方便切换使用的 npm 源切换工具

# 安装

npm i -g nrm
1

# 常用命令

查看当前版本

  nrm -V
1

查看当前支持的源

  nrm ls
1

修改当前 npm 指定源

  nrm use [name]
