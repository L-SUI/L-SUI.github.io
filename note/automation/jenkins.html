# Download and install

长期支持版本 (LTS)。下载安装包，不要下载war格式的。

服务器可下载rpm格式的然后上传。用rpm命令直接解压完成即可。

下载两种方式可以看这篇文章

rpm -ivh jenkin-**.rpm

rpm -qa jenkins
#会输出一个Jenkins**.noarch文件
systemctl start jenkins
#启动Jenkins 打开浏览区访问服务器域名8080端口
#Jenkins安装完成之后是一个服务，不用的时候可以关掉。
